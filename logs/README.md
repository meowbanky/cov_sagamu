# Email Queue Logs

This directory contains log files for the email queue system.

## Files:
- `email_queue.log` - Main email queue processing logs

## Log Rotation:
Logs are automatically cleaned up by the cron job to prevent disk space issues.

## Access:
Logs can be viewed via:
- SSH access to server
- Email queue dashboard (if implemented)
- Server file manager
